Research

Payment Intelligence Systems: A Machine Learning Approach to Fraud Detection
Sabari G, et al.
We present a novel machine learning framework for real-time fraud detection in payment processing systems. Our approach combines ensemble methods with deep learning architectures to achieve high accuracy while maintaining low latency. The system has been deployed in production and processes millions of transactions daily with a false positive rate below 0.1%.
Retrieval-Augmented Generation for Financial Document Analysis
Sabari G, et al.
This work explores the application of RAG (Retrieval-Augmented Generation) architectures for analyzing financial documents. We demonstrate how vector databases combined with large language models can provide accurate, context-aware responses for financial queries. The system uses pgvector for efficient similarity search and Ollama for local LLM inference, achieving state-of-the-art performance on financial Q&A tasks.
Scalable ML Pipelines for Fintech Applications
Sabari G, et al.
We describe the architecture and implementation of production-grade machine learning pipelines for financial technology applications. The system leverages containerization, orchestration tools, and cloud infrastructure to enable rapid model deployment and scaling. Key contributions include automated model versioning, A/B testing frameworks, and real-time monitoring capabilities.
GenAI Copilots for Financial Automation
Sabari G, et al.
This paper introduces a framework for building GenAI-powered copilots that assist in financial decision-making and automation tasks. We explore how large language models can be fine-tuned and integrated with financial data systems to provide intelligent assistance while maintaining security and compliance standards. The system has been successfully deployed in production environments.
